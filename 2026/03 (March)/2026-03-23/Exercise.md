# No Consecutive Repeats

https://www.freecodecamp.org/learn/daily-coding-challenge/2026-03-23

Given a string, determine if it has no repeating characters.

- A string has no repeats if it does not have the same character two or more times in a row.

## Tests

1. `has_no_repeats("hi world")` should return `True`.
1. `has_no_repeats("hello world")` should return `False`.
1. `has_no_repeats("abcdefghijklmnopqrstuvwxyz")` should return `True`.
1. `has_no_repeats("freeCodeCamp")` should return `False`.
1. `has_no_repeats("The quick brown fox jumped over the lazy dog.")` should return `True`.
1. `has_no_repeats("Mississippi")` should return `False`.
